# Slack Thread Copier Pro

Slackのスレッド全文を**ワンクリックでコピー**できるChrome拡張機能です。  
**AIチャット、AIスキル、議事録作成、他ツールへの転記**などに最適です。

---

## 背景と目的

重要な議論は **Slackスレッド** 上で行われることが多く、その内容を **AIチャット（ChatGPT、Claudeなど）やAIスキル** に入力して、次のような用途に活用するケースが増えています。

- 要約
- 分析
- 論点整理
- アクションアイテムの抽出

しかしSlackは **virtual scroll UI** を採用しているため、長いスレッドではすべてのメッセージがDOM上に存在するわけではありません。  
そのため、ユーザーは **何度もスクロールしながらメッセージを手動でコピーする必要** があり、手間がかかるうえ、内容の取りこぼしが発生する可能性もあります。

この拡張機能は、その作業を **ワンクリック** に削減します。

---

## 主な機能

### Copy ボタン

メッセージにホバーすると **Copy** ボタンが表示され、単一メッセージをコピーできます。

---

### Thread ボタン / Copy Full Thread

スレッドペイン内の **Thread** ボタン、またはヘッダーの **Copy Full Thread** ボタンを使用すると、スレッド全文をコピーできます。

特徴：

- Slackの **virtual scroll** に対応
- スレッドを自動スクロールして全メッセージを収集
- 取得中は進捗を表示（例：`42 msgs…`）

---

### フォーマット切り替え（Popup設定）

拡張機能アイコンをクリックすると設定画面が開き、コピー形式を選択できます。

| 形式 | 出力例 | 用途 |
|------|------|------|
| **Plain Text**（デフォルト） | `[John Doe]\nメッセージ内容` | AIチャット / AIスキル |
| **Markdown** | `**John Doe**\nメッセージ内容` | ドキュメント / 議事録 |

---

### Prefix 設定

コピー内容の先頭に固定テキストを追加できます。

**例：**


@skill-name
以下のSlackスレッドを読んで、議論のポイント、課題、アクションアイテムを整理してください。


設定したPrefixは保存され、次回以降も自動で付加されます。

---

## インストール方法

1. このリポジトリをダウンロード（または `git clone`）
2. Chromeで `chrome://extensions` を開く
3. 右上の **Developer mode** をONにする
4. **Load unpacked** をクリック
5. `manifest.json` が含まれるフォルダを選択

インストール後、`app.slack.com` を開いて拡張機能が有効になっていることを確認してください。

---

## 使い方

### メッセージをコピーする

1. Slackでメッセージにホバー
2. 表示される **Copy** ボタンをクリック
3. メッセージがクリップボードにコピーされます

---

### スレッド全文をコピーする

1. Slackでスレッドを開く
2. **Thread** ボタン（メッセージホバー時）または **Copy Full Thread**（スレッドヘッダー）をクリック
3. 自動スクロールして全メッセージを収集
4. 完了後、AIチャットやドキュメントに貼り付け

---

### 設定を変更する

1. ツールバーの拡張機能アイコンをクリック
2. **Copy Format**（PlainText / Markdown）を選択
3. **Prefix** に追加したい文章を入力
4. 設定は自動保存されます
